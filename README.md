# GLSL Raymarcher

I made a basic 2D and 3D raymarcher in GLSL with OpenGL and C++ for fun.

## Controls

### 2D:

|    Input     |        Action        |
|:------------:|:--------------------:|
|   **WASD**   |         Move         |
|**RMB + Drag**|         Move         |
|   **LMB**    |Show distance to mouse|
|  **Scroll**  |         Zoom         |

### 3D:

|    Input     |         Action         |
|:------------:|:----------------------:|
|   **WASD**   |Move on horizontal plane|
|**Left Shift**|       Move down        |
|   **Space**  |        Move up         |
|**RMB + Drag**|     Rotate camera      |
